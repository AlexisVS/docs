## Technology Stack

AGVES is built with modern, production-ready technologies:

### Frontend
- **Next.js 15** with App Router and React Server Components
- **React 19** with latest hooks and concurrent features
- **TypeScript** with strict configuration for type safety
- **TailwindCSS** + **Material-UI** for consistent styling
- **React Query** for server state management

### Backend
- **Laravel 11** API with auto-generated TypeScript types
- **MySQL** database with optimized entity relationships
- **Laravel Sanctum** for API authentication
- **Laravel Broadcasting** for real-time features

### Development Tools
- **Vitest** for unit and integration testing
- **Playwright** for end-to-end testing
- **Docker** for containerized development
- **Laravel Pint** for code formatting
- **PHPStorm** / **VS Code** for IDE support

## Core Features

### Entity-Driven Architecture
- **5 business modules** with 22 entities
- **Automatic CRUD** generation for all entities
- **Type-safe relationships** between entities
- **Dynamic form generation** based on entity schemas

### Multi-Language Support
- **3 languages** supported: English, French, Dutch
- **Entity-aware translations** for business terms
- **RTL support** for Arabic and Hebrew markets
- **Dynamic language switching**

### Modern Developer Experience
- **Hot reload** in development with Vite
- **Type generation** from Laravel to TypeScript
- **Component documentation** with Storybook
- **API documentation** with OpenAPI integration

## Contributing

We welcome contributions! Please see our [Contributing Guide](https://github.com/yourusername/agves_app/blob/main/CONTRIBUTING.md) for details.

### Development Setup
```bash
# Clone the repository
git clone https://github.com/yourusername/agves_app.git

# Install dependencies
npm install && composer install

# Setup environment
cp .env.example .env

# Generate application key
php artisan key:generate

# Run migrations
php artisan migrate

# Start development servers
npm run dev & php artisan serve
```
